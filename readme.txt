This plugin extends WooCommerce with Mastercard Payment Gateway Services.

== Description ==
Mastercard Payment Gateway Services for WooCommerce is a payment gateway plugin that allows you to take credit card payments via MPGS directly on your WooCommerce site.

== Installation ==

1. Download the WooCommerce MPGS plugin zip file and extract to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce backend settings page
4. Navigate to Payments tab
5. Then go to settings of MPGS Online Payment Gateway. Fill the form with credentials and save changes then you are ready.
